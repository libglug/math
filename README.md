# math
3d maths

### Building
```
mkdir build && cd build
cmake [path/to/repo]
cmake --build . --target glug_math
```

### Running tests
```
mkdir build && cd build
cmake [path/to/repo]
cmake --build . --target glug_math_tests
cd test
./glug_math_tests
```
